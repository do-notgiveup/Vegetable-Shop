# Java Web application

## Features:

- Login and logout.
- Search product by name
- Create, update, delete product as admin.
- Add product to cart.
- Edit cart.
- Delete product from cart.
- Checkout.

## Tech:

- Java servlet
- CRUD

Project use:

- [Java]
- [Tomcat]
- [MySQL]
